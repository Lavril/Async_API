## Movies Admin (Django админка)

`movies_admin` — Django‑приложение админки для управления контентом кинотеатра.

В Docker запускается через **uWSGI**, статические файлы собираются командой `collectstatic` на этапе сборки образа.

---

## Запуск через Docker Compose (рекомендуется)

Админка поднимается **из корня репозитория** вместе с БД и nginx:

```bash
cd ..
cp .env.example .env
docker compose up --build
```

---

## Полезно знать

- Если админка «не открывается», проверьте, что:
  - подняты `movies-admin`, `theatre-db`, `nginx`,
  - переменные окружения в `.env` корректны.
